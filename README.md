# Taskify

A task management application with fully secured authentication & authorization.

## Overview

Taskify is a comprehensive task management application built with modern web technologies, featuring robust security measures including fully secured authentication and authorization mechanisms to protect user data and ensure secure access control.

## Features

- ✅ **Secure Authentication** - Fully secured user authentication system
- ✅ **Authorization & Access Control** - Fine-grained authorization to manage user permissions
- ✅ **Task Management** - Create, update, and organize tasks efficiently
- ✅ **User-Friendly Interface** - Intuitive and responsive design
- ✅ **Data Security** - Protected user data with industry-standard security practices

## Technology Stack

- **Backend**: PHP (74.3%)
- **Frontend**: HTML (24.2%), CSS (0.4%)
- **Additional**: Hack (1.1%)
